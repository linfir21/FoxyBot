# FoxBot

Telegram-бот, который отправляет случайные фотографии лис.

## Особенности

- Отправляет случайные фото лис через API [randomfox.ca](https://randomfox.ca/)
- При недоступности основного API автоматически переключается на фото котов (fallback)
- Улучшенное логирование с ротацией файлов (`foxbot.log`)
- Простая клавиатура с кнопкой `/newfox`

## Команды

| Команда  | Описание                                    |
|----------|---------------------------------------------|
| `/start` | Приветствие и первая случайная фотография   |
| `/newfox`| Отправить новое случайное фото лисы         |

## Установка и запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```


2. Запустите бота:
   ```bash
   python foxbot.py
   ```

## Структура проекта

```
foxbot/
├── foxbot.py         # Основной файл бота
├── requirements.txt  # Зависимости проекта
└── README.md         # Описание проекта
```

## Логирование

Логи сохраняются в файл `foxbot.log`. При достижении размера 50 МБ файл автоматически архивируется (сохраняется до 5 резервных копий).
